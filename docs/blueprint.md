# **App Name**: DopamineLab

## Core Features:

- Daily Challenges: Presents users with daily challenges, each consisting of a question with multiple choices. Includes a countdown timer and displays vote counts/percentages in real-time.
- Community Polls: Allows users to create and participate in polls on various topics. Users can vote only once per poll, and the system updates results in real-time, displaying vote counts and percentages.
- Prediction System: Enables users to make predictions on future events with specified timeframes. Other users can vote on these predictions, fostering daily engagement.
- User-Generated Content (UGC): Users can submit their own polls, challenges, and predictions, subject to filtering to prevent spam and offensive content.
- Comment System: Allows users to comment on polls and predictions, with upvote/downvote functionality. AI will analyze and pin the highest-rated comment to foster better discussions and community engagement. Tool.
- Addictive Loop: Implements a feedback mechanism that utilizes the publication of a new question on some repeating interval. The platform provides immediate feedback after voting and social comparison features. 

## Style Guidelines:

- Primary color: Neon purple (#C271D4) for a modern and youthful feel.
- Background color: Dark purple (#221429), a desaturated variant of the primary.
- Accent color: Neon blue (#71B4D4) to complement the primary, used for interactive elements.
- Body and headline font: 'Space Grotesk' for headlines, paired with 'Inter' for body text. Note: currently only Google Fonts are supported.
- Use simple, modern icons that align with the youthful style of the application.
- Incorporate subtle animations and transitions to enhance user experience and engagement.
- Dark mode design with card-based sections for easy navigation and a visually appealing interface.
